プライバシーポリシー

最終更新日: 2025年2月21日

Sofvo（以下「当サービス」）は、ユーザーの個人情報の保護を重要視しています。本プライバシーポリシーは、当サービスにおける個人情報の取り扱いについて説明します。

1. 収集する情報

当サービスでは、以下の情報を収集する場合があります。

1.1 ユーザーが提供する情報

情報の種類利用目的
メールアドレスアカウント認証・重要なお知らせ
ニックネーム・プロフィール情報サービス内での表示・マッチング
プロフィール画像サービス内での表示
投稿内容（テキスト・画像）コミュニティ機能の提供
チャットメッセージコミュニケーション機能の提供
大会情報大会運営機能の提供

1.2 自動的に収集される情報

  • 端末情報（OS、バージョン）
  • 利用状況データ（アクセス日時、利用機能）

2. 情報の利用目的

収集した情報は、以下の目的で利用します。

  1. 当サービスの提供・運営・改善
  2. ユーザーからのお問い合わせへの対応
  3. 利用規約に違反する行為への対応
  4. 当サービスに関する重要なお知らせ
  5. 新機能・更新情報のご案内
  6. 統計データの作成（個人を特定できない形式）

3. 情報の共有・第三者提供

当サービスは、以下の場合を除き、ユーザーの個人情報を第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合
  • サービス提供に必要な業務委託先への提供（Firebase / Google Cloud 等）

4. 外部サービスの利用

当サービスでは、以下の外部サービスを利用しています。

サービス名提供元利用目的
Firebase AuthenticationGoogle LLCユーザー認証
Cloud FirestoreGoogle LLCデータベース
Firebase StorageGoogle LLC画像・ファイル保存
Firebase Cloud MessagingGoogle LLCプッシュ通知

各サービスのプライバシーポリシーについては、それぞれの提供元のWebサイトをご確認ください。

5. データの保管とセキュリティ

  • ユーザーデータはGoogle Cloudのインフラ上に暗号化して保管されます。
  • Firebase Security Rulesにより、ユーザーは自身のデータのみアクセス可能です。
  • パスワードはFirebase Authenticationにより安全にハッシュ化されています。

6. ユーザーの権利

ユーザーは以下の権利を有します。

  1. データの閲覧：アプリ内のマイページからご自身の情報を確認できます。
  2. データの修正：プロフィール編集画面から情報を修正できます。
  3. データの削除：設定画面の「アカウント削除」からデータを完全に削除できます。
  4. 通知の管理：設定画面から通知の受信設定を変更できます。

7. Cookie等の使用

当サービスのWeb版では、認証情報の保持のためにブラウザのローカルストレージを使用する場合があります。

8. 子どものプライバシー

当サービスは13歳未満のお子様を対象としていません。13歳未満のお子様から個人情報を収集していることが判明した場合、速やかに該当情報を削除します。

9. プライバシーポリシーの変更

当サービスは、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、当サービス上でお知らせします。変更後のポリシーは、当ページに掲載された時点から効力を生じます。

10. お問い合わせ

プライバシーに関するお問い合わせは、お問い合わせフォームからご連絡ください。
