# 開発ロードマップ

## Phase 5：高優先度（次回作業）
### PDFダウンロード3種
- 大会要項PDF（大会名・日程・会場・ルール・スケジュール・注意事項）
- 対戦表PDF（コート別試合一覧・審判チーム・スコア）
- トーナメント表PDF（ブラケット形式・スコア・順位表）
- 日本語フォント（Noto Sans JP）バンドル必要

### 大会スケジュール設定・表示
- 開場、受付開始、代表者ミーティング、開会式、試合開始、昼休憩、決勝開始、大会終了予定、完全撤収
- 概要タブにタイムライン形式で表示
- PDFにも自動反映

### 勝ち点計算の精度向上
- 2-0勝利=10点, 1-1得失点差勝ち=7点, 1-1引き分け=4点, 得失点差負け=2点, 0-2敗北=0点

## Phase 5：高優先度（その次）
### エントリー締切自動化
- 大会作成時に締切日時を設定
- 締切後は自動でエントリー停止

### QRコード受付
- 3パターン選択可能: チーム代表者読み取り / 主催者読み取り / 受付なし（手動）
- 管理画面でリアルタイム到着状況確認
- 全チーム到着後「大会を開始する」ボタン表示

### その他高優先度
- 参加者向け画面整備（非主催者の試合・結果・順位表示）
- 予選2の動作確認（順位ベースのコート再編成）
- 大会終了後の結果画面（最終順位発表）
