# Sofvo - アプリ概要

## バージョン: 5.0
## 最終更新: 2026-02-17
## URL: https://sofvo-19d84.web.app

## 概要
バレーボール大会運営・SNSアプリ。大会の作成からエントリー、対戦表自動生成、スコア入力、順位集計、決勝トーナメントまでワンストップで管理。SNS機能でチーム間の交流も促進。

## 技術スタック
- フレームワーク: Flutter Web (Dart)
- バックエンド: Firebase (Auth / Firestore / Storage / Hosting)
- 将来検討: Stripe決済, Google Maps, FCM, OpenWeatherMap API
