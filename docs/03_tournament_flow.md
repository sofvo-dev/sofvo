# 大会運営フロー

## 9ステップ
1. 大会作成（ルール・会場・スケジュール設定）
2. チームエントリー（締切日時で自動締切）
3. 対戦表自動生成（ラウンドロビン・審判チーム割り当て）
4. 予選1（コート別総当たり）
5. コート別順位集計
6. 予選2（順位ベースでコート再編成）
7. 総合順位決定
8. 決勝トーナメント（上位/下位）
9. 結果発表

## スコア入力仕様
- セット終了後に最終スコアを数字入力
- スライドで主審確認 → 各チーム監督確認
- 入力中は自動保存（戻っても消えない）
- 15点先取・ジュースあり時は2点差またはキャップ（17点）で決着

## バリデーションルール
- ジュースなし: 勝者15点、敗者14点以下
- ジュースあり（通常）: 15点以上で2点差
- ジュースあり（デュース突入）: 両者14点以上で2点差
- ジュースあり（キャップ）: 勝者17点、敗者16点

## 勝ち点表（2セットマッチ）
| 結果 | 勝ち点 |
|------|--------|
| 2-0 勝利 | 10点 |
| 1-1 得失点差勝ち | 7点 |
| 1-1 引き分け | 4点 |
| 1-1 得失点差負け | 2点 |
| 0-2 敗北 | 0点 |

順位決定: 勝ち点 → 得失点差 → 総得点 → 直接対決
